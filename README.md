# react-native-web

## To set up:
```
yarn install
```
## To run the development server for web app:
```
cd ./packages/web
yarn start
```
## To run the development server for rn app:
```
cd ./packages/app
```
###### delete the react folder from node_modules
```
yarn start
yarn android
```
## To run storybook app:
```
yarn storybook
yarn start
yarn android
```
## To apply changes for common components:

```
yarn build
```
(in ./packages/common)
